1) .env.local dosyasını proje kökünde oluştur ve Firebase + davet kodunu yaz.
2) npm install
3) npm run dev
4) /  -> davet
   /login -> giriş (Google + Email)
   /register -> kayıt
   /dashboard -> guard'lı alan
5) Vercel deploy -> aynı env değişkenlerini Project Settings > Environment Variables'a ekle, redeploy.
